# 手记视图插件 v0.1.0

## 安装说明

### 必需文件（3个）

本目录包含插件运行所需的三个核心文件：

1. **`manifest.json`** - 插件元数据
2. **`main.js`** - 编译后的主程序
3. **`styles.css`** - 样式表

### 安装步骤

1. **复制文件到 Obsidian 插件目录**
   - 打开 Obsidian
   - 进入 `设置` → `第三方插件` → `已安装的插件`
   - 点击文件夹图标打开插件目录
   - 创建新文件夹 `obsidian-journal-view`
   - 将本目录中的 3 个文件复制到该文件夹

2. **启用插件**
   - 在 Obsidian 设置中，找到 `第三方插件`
   - 找到 `手记视图` 插件
   - 启用插件

3. **使用插件**
   - 使用命令面板（`Ctrl+P` / `Cmd+P`）
   - 输入 `打开手记视图`
   - 或通过右键菜单创建文件夹手记视图

### 系统要求

- Obsidian 版本 >= 0.15.0
- 支持桌面端和移动端

### 功能特性

- 📅 按日历组织 Markdown 文件
- 🖼️ 从正文中提取图片显示
- 📊 按月份分组显示
- 🎨 现代化卡片式布局
- ⚡ 高性能分页加载
- 🎯 右键菜单快速创建视图
